My name is Ryan Asher and I am a programmer. I enjoy coding and videogames. Sometimes I even try to code videogames.

Projects

  • Pizza Tracker: A static page that records the slices of pizza my wife and I eat while we live in New York City. Pizza slice totals are stored in SimpleDB, and retrieved via an API Gateway endpoint that runs a Lambda function that accesses the DB. A separate submit page exists that uses similar logic, so Sharon and I can update the totals from our mobile devices.

  • Ryanflix: A simple tool that parses JSON episode data and selects a random episode from some of my favorite TV shows.

  • Square Balls: My first independently created videogame with "art" (!). Square Balls has you controlling a square spaceship and avoiding the evil circle and triangle spaceships. Playable for up to 4 people.

  • The Sweeping of iVac: Afterdirt+: Our videogame submission for Ludum Dare 37. Made in GameMaker with the help of Sebastian Karolkiewicz (art) and Patrick O'Brien (sound).

  • Text-based Games: Some text games I have made in Python. Includes "classic hits" like Bomb Defusal and Data Breach.

  • TV Episode Scraper: A TV episode scraper, which allows you to pull the episode data of your favorite shows.
